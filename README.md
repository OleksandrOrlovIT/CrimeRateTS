## Description
Nodejs typeScript nestJS project

## Installation

```bash
$ npm install

$ docker compose up
//to start a mongodb instance

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
#tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
